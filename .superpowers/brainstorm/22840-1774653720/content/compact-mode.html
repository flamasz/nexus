How should the dropdown panel open in row context?

The search/select dropdown needs to appear somewhere when clicked

A

Floating dropdown below the field

Dropdown opens directly below the compact trigger, floating over other content. Same dropdown panel as the modal version, just positioned absolutely. This is how the existing CategorySelector already works (portal-rendered).

Consistent with existing pattern. May overlap other row items — standard UX for dropdowns.

B

Popover/modal overlay

Clicking the field opens a small centered popover or modal with the full search+select experience. Provides more room for the search list and "create new" form without competing for row space.

More disruptive but gives more room. Better for complex fields like Category with dimensions.
